# Office Project

你当前在 **Office 文档项目** 目录中。本目录包含 **7 个领域 Skills**，覆盖 PDF、Word、Excel、PowerPoint 文档处理和 LaTeX 海报制作。

## 使用方式

- 用 `/skill-name` 直接调用，或描述需求自动匹配
- 全局流程 Skills（brainstorming、TDD、debugging 等）始终可用
- 本地 Skills 按需加载，不会一次性全部载入上下文

## 可用 Skills

| Skill | 用途 |
|-------|------|
| `pdf` | PDF 读取、提取、转换 |
| `docx` | Word 文档生成和编辑 |
| `xlsx` | Excel 数据处理和生成 |
| `pptx` | PowerPoint 演示文稿 |
| `pptx-posters` | PPT 海报设计 |
| `latex-posters` | LaTeX 学术海报 |
| `markitdown` | Markdown 文档转换 |
