# Database Project

你当前在 **数据库查询项目** 目录中。本目录包含 **23 个领域 Skills**，覆盖文献数据库、生物医学数据库、金融数据库和公共数据集查询。

## 使用方式

- 用 `/skill-name` 直接调用，或描述需求自动匹配
- 全局流程 Skills（brainstorming、TDD、debugging 等）始终可用
- 本地 Skills 按需加载，不会一次性全部载入上下文

## 核心领域

| 方向 | 代表 Skills |
|------|------------|
| 文献 | `pubmed-database`, `openalex-database` |
| 生物医学 | `uniprot-database`, `pdb-database`, `chembl-database`, `clinvar-database` |
| 基因组 | `gene-database`, `ensembl-database`, `geo-database` |
| 金融/商业 | `alpha-vantage`, `fred-economic-data`, `edgartools` |
| 专利/药物 | `uspto-database`, `drugbank-database`, `fda-database` |
