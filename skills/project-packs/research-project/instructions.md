# Research Project

你当前在 **研究项目** 目录中。本目录包含 **24 个领域 Skills**，覆盖论文写作、文献综述、基金申请、同行评审和科学可视化。

## 使用方式

- 用 `/skill-name` 直接调用，或描述需求自动匹配
- 全局流程 Skills（brainstorming、TDD、debugging 等）始终可用
- 本地 Skills 按需加载，不会一次性全部载入上下文

## 核心领域

| 方向 | 代表 Skills |
|------|------------|
| 写作 | `scientific-writing`, `literature-review`, `research-en`, `research-zh` |
| 基金 | `research-grants`, `venue-templates` |
| 评审 | `peer-review`, `scholar-evaluation` |
| 可视化 | `scientific-visualization`, `scientific-schematics`, `scientific-slides` |
| 临床 | `clinical-decision-support`, `clinical-reports`, `treatment-plans` |
