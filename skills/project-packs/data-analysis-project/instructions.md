# Data Analysis Project

你当前在 **数据分析项目** 目录中。本目录包含 **24 个领域 Skills**，覆盖探索性分析、统计建模、可视化、地理分析和时间序列预测。

## 使用方式

- 用 `/skill-name` 直接调用，或描述需求自动匹配
- 全局流程 Skills（brainstorming、TDD、debugging 等）始终可用
- 本地 Skills 按需加载，不会一次性全部载入上下文

## 核心领域

| 方向 | 代表 Skills |
|------|------------|
| EDA | `exploratory-data-analysis`, `polars`, `dask` |
| 可视化 | `matplotlib`, `seaborn`, `plotly`, `infographics` |
| 统计 | `statistical-analysis`, `statsmodels`, `pymc`, `scikit-survival` |
| 地理 | `geopandas`, `geomaster` |
| 预测 | `timesfm-forecasting`, `simpy` |
