# Scientific Project

你当前在 **科学计算项目** 目录中。本目录包含 **46 个领域 Skills**，覆盖单细胞分析、分子模拟、深度学习、生物信息学和化学信息学。

## 使用方式

- 用 `/skill-name` 直接调用，或描述需求自动匹配
- 全局流程 Skills（brainstorming、TDD、debugging 等）始终可用
- 本地 Skills 按需加载，不会一次性全部载入上下文

## 核心领域

| 方向 | 代表 Skills |
|------|------------|
| 单细胞 | `scanpy`, `anndata`, `scvi-tools`, `scvelo` |
| 分子/化学 | `rdkit`, `datamol`, `deepchem`, `medchem`, `molecular-dynamics` |
| 蛋白质/基因 | `esm`, `biopython`, `pysam`, `pydeseq2` |
| ML/DL | `pytorch-lightning`, `transformers`, `scikit-learn`, `shap` |
| 模拟/材料 | `pymatgen`, `pyscf`, `stable-baselines3` |
