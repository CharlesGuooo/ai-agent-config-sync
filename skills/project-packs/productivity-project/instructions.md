# Productivity Project

你当前在 **生产力工具项目** 目录中。本目录包含 **24 个领域 Skills**，覆盖 Obsidian、Jira、Google Workspace、项目管理和团队协作。

## 使用方式

- 用 `/skill-name` 直接调用，或描述需求自动匹配
- 全局流程 Skills（brainstorming、TDD、debugging 等）始终可用
- 本地 Skills 按需加载，不会一次性全部载入上下文

## 核心领域
